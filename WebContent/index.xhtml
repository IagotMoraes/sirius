S.I.RI.US - Sistema de Identificação de Riscos Universais

O sistema de identificação de riscos universais, S.I.RI.US, é um plataforma que busca facilitar a identificação de riscos dentro de projetos através de suas caracteristiscas, realizando calculos probabilisticos para indicar quais são as principais preocupações dentro do projeto.
